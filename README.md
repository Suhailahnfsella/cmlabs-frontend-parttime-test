## 📌 Description

Aplikasi web berbasis Vue.js untuk menampilkan data makanan berdasarkan ingredient.

Fitur utama:

- Menampilkan daftar ingredients
- Pencarian ingredient
- Menampilkan daftar meal berdasarkan ingredient
- Pencarian meal
- Detail meal (gambar, instruksi, bahan, dan video)
- Pagination (termasuk opsi "All")

## 📂 Project Setup

### 1. Clone Repository

```bash
git clone https://github.com/Suhailahnfsella/cmlabs-frontend-parttime-test.git
cd cmlabs-frontend-parttime-test
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Setup Environment

Buat file `.env` di root project:

```env
VITE_API_BASE_URL=https://www.themealdb.com/api/json/v1/1
```

### 4. Run Project

```bash
npm run dev
```

### 5. Build Project

```bash
npm run build
```
